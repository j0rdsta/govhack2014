Aim

The aim for this project is to design a web application that enables city goers quick access to the locations of important areas or services. This way, the user can quickly find a public service or location.

Future Uses and Expansions

This application could be expanded by creating a mobile application. This would allow user to bring their City Companion with them as they explore the city.

Jake was able to create a live wireframe demonstration which can be viewed here.

Development Notes

In this project we used PHP, Laravel 4.1 (a PHP framework), MySQL, Zurb Foundation 5, Javascript, jQuery and the Google Maps Javascript API.

Data Sets

This project uses City of Gold Coast datasets that include:

  • Public Barbecues
  • Public Showers
  • Bike Racks
  • Fish Cleaning Stations
  • Boat Ramps
  • Drinking Fountains
  • Fitness Equipment
  • Dog Bag Dispensers
  • Outdoor Taps
  • Road Closures
  • Cycleways
  • Picnic Tables
